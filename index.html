Component Functions

Component Function/Use
ESP32 Main microcontroller that reads sensor data, connects to Wi-Fi, sends data to Firebase, and enters deep sleep mode to conserve power.
DHT22 Sensor Measures ambient temperature and humidity. Data is used for environmental monitoring and analysis.
Capacitive Soil Moisture Sensor (Analog) Reads soil moisture levels via analog input. The values are calibrated to determine dryness or wetness for plant hydration status.
Water Level Float Switch (Digital) Detects whether the water level in the tank is LOW or HIGH using a simple float mechanism and digital input pin.
Firebase Realtime Database Stores both the latest and historical sensor data. Enables remote monitoring via web or mobile dashboards.
Wi-Fi (ESP32 built-in) Used to connect the ESP32 to the internet and send sensor readings to Firebase.
Deep Sleep Mode Reduces power consumption by putting ESP32 into low-power mode for 5 minutes between sensor readings.
